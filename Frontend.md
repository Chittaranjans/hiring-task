# Home Assessment: Frontend Developer

## Objective

We are looking for a Frontend Developer to create a web application that interacts with a blockchain using Web3 technologies. This assessment will evaluate your skills in React, TypeScript, and your understanding of Web3 concepts.

## Assessment Overview

You will develop a simple decentralized application (dApp) that allows users to connect their cryptocurrency wallet, view their wallet balance, and display a list of tokens they hold. The backend for authentication will be provided via an Express server.

## Requirements

1. **Frontend Development:**
   - Use React and TypeScript to build the user interface.
   - Implement a responsive design that works on both desktop and mobile devices.

2. **Web3 Integration:**
   - Utilize Web3.js or Ethers.js to interact with the blockchain.
   - Allow users to connect their wallet (e.g., MetaMask).
   - Fetch and display the user's wallet balance and token holdings.

3. **Authentication:**
   - Integrate with the provided Express backend for user authentication.
   - All frontend interactions must occur after user authentication.

4. **Documentation:**
   - Provide clear documentation on how to set up and run your application.
   - Include comments in your code to explain your logic and decisions.

## Evaluation Criteria

- Code quality and organization
- Use of best practices in React and TypeScript
- Understanding and implementation of Web3 concepts
- User experience and design
- Clarity and completeness of documentation
